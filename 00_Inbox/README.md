# 00_Inbox — 收集

暫存所有尚未處理的想法、任務、請求或素材。這裡只負責「收」，不負責「做」。

不肯定某份 file 該放哪裡時，先放這裡，再叫 agent 幫你分流（見 `../Folder_Routing_Map.md`）。

## 使用方式
- 每個項目一個檔案，或用一份清單檔累積都可以，先求記錄不遺漏
- 定期（例如每天/每次 session）檢視 inbox，把該處理的項目移到 `../01_Action_Center/`，或整理成參考資料放進 `../02_Knowledge_Base/`
- 處理完的項目從這裡移除，避免 inbox 越堆越多

## 注意
內容不得包含密碼、API key、私人資料、薪酬資料、員工紀錄或客戶紀錄。
