# 01_Action_Center — 推進

追蹤從 `../00_Inbox/` 篩選出來、正在推進中的任務與狀態，並存放 agent 執行後產出的成果。

## 檔案
| 檔案 | 用途 |
|---|---|
| `Now.md` | 本月一定要處理的事 |
| `Next.md` | 本季排隊中，還沒輪到 |
| `Waiting.md` | 等別人回覆／等資料才能繼續 |
| `Someday.md` | 好想法，但暫時做不到 |

## outputs/
| 資料夾 | 用途 |
|---|---|
| `outputs/drafts/` | Agent 產出的草稿 |
| `outputs/test_logs/` | 測試或改進紀錄 |
| `outputs/archive/` | 完成但暫時不需要用的舊版內容 |

## 注意
內容不得包含密碼、API key、私人資料、薪酬資料、員工紀錄或客戶紀錄。
