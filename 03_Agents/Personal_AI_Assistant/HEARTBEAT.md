# HEARTBEAT

> 每天固定的節奏：上班開始與收工前分別要做什麼。

## Daily Routine

### Start（08:30）
1. Calendar & Agenda Sync — 檢視今天行程、標記高優先會議、確認 buffer window
2. High-Alert Triage — 檢查隔夜緊急事件/bug，轉交對應團隊
3. Pipeline & Deliverable Status Update — 追蹤未完成事項，列出今天前 3 大優先任務
4. 用以下範本傳晨間狀態訊息給我：

> Good morning, Sisi. Today's Schedule: [...] / Urgent Matters: [...] / Pending Decisions Needed From You: [...] / My Focus for Today: [...]

### End（19:00）
1. 30-Minute Status Reconciliation — 檢查明天行程、確認 buffer window、準備好明早會議所需資料
2. Action-Item & Deliverable Governance — 更新跨部門待辦狀態，標記錯過的截止日
3. High-Alert & Escalation Triage — 檢視今天的爭議/bug，確認已登記工單
4. 用以下範本傳晚間狀態訊息給我：

> Good evening, Sisi. Today's Key Achievements: [...] / Outstanding/Carried Over Tasks: [...] / Tomorrow's High-Priority Overview: [...]
